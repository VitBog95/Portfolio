# Баг-репорт №4: Краткое описание фильма вместо развёрнутого

**Название:** Описание фильма представлено в виде краткой аннотации  
**Приоритет:** Low  
**Серьёзность:** Trivial  

**Окружение:**
- Браузер: Chrome 146.0.7680.165 (64-bit)
- ОС: Windows 11
- Сайт: https://www.kinopoisk.ru/

**Шаги воспроизведения:**
1. Открыть главную страницу Кинопоиска.
2. Выбрать любой фильм (например, «Князь Андрей»).
3. Перейти на страницу фильма.
4. Обратить внимание на текстовое описание под названием.

**Фактический результат:**  
Под названием отображается только краткая аннотация (одно предложение), отсутствует развёрнутое описание.

**Ожидаемый результат:**  
Есть полное описание фильма (сюжет, режиссёр, актёры и т.д.).
