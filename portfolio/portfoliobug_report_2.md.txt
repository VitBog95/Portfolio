# Баг-репорт №2: Невозможно применить комбинированную фильтрацию из-за отсутствия фильтра по рейтингу

**Название:** Невозможно применить комбинированную фильтрацию с участием рейтинга  
**Приоритет:** High  
**Серьёзность:** Major  

**Окружение:**
- Браузер: Chrome 146.0.7680.165 (64-bit)
- ОС: Windows 11
- Сайт: https://www.kinopoisk.ru/

**Шаги воспроизведения:**
1. Открыть главную страницу Кинопоиска.
2. Нажать на кнопку «Фильтры».
3. Установить фильтр по году: 2019–2020.
4. Выбрать жанр: «аниме».
5. Попытаться выбрать рейтинг: «8+».

**Фактический результат:**  
На шаге 5 обнаруживается, что фильтра по рейтингу нет. Комбинированная фильтрация с участием рейтинга невозможна.

**Ожидаемый результат:**  
Фильтр по рейтингу присутствует, пользователь может применить его вместе с годом и жанром, результаты поиска соответствуют всем условиям.

**Связанные баги:** Баг-репорт №1
