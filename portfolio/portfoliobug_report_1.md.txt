# Баг-репорт №1: Отсутствие фильтрации по рейтингу в панели фильтров

**Название:** Отсутствие блока «Рейтинг» в панели фильтров  
**Приоритет:** High  
**Серьёзность:** Major  

**Окружение:**
- Браузер: Chrome 146.0.7680.165 (64-bit)
- ОС: Windows 11
- Сайт: https://www.kinopoisk.ru/

**Шаги воспроизведения:**
1. Открыть главную страницу Кинопоиска.
2. Нажать на кнопку «Фильтры» (рядом с полем поиска).
3. В открывшейся панели прокрутить список параметров.
4. Визуально найти блок «Рейтинг».

**Фактический результат:**  
Блок «Рейтинг» отсутствует. Нет возможности отфильтровать фильмы по рейтингу (например, 7+, 8+).

**Ожидаемый результат:**  
В панели фильтров присутствует блок «Рейтинг» с возможностью выбора минимального рейтинга.
