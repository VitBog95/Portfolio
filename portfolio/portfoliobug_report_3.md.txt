# Баг-репорт №3: Отсутствует кнопка «Сбросить фильтры» в панели фильтров

**Название:** Отсутствует кнопка «Сбросить фильтры» в панели фильтров  
**Приоритет:** High  
**Серьёзность:** Major  

**Окружение:**
- Браузер: Chrome 146.0.7680.165 (64-bit)
- ОС: Windows 11
- Сайт: https://www.kinopoisk.ru/

**Шаги воспроизведения:**
1. Открыть главную страницу Кинопоиска.
2. Нажать на кнопку «Фильтры».
3. Установить фильтры: год 2018, жанр «Комедия», страна «Россия».
4. Визуально найти элемент для сброса всех фильтров.

**Фактический результат:**  
Кнопка «Сбросить фильтры» или аналогичный элемент отсутствует.

**Ожидаемый результат:**  
Присутствует кнопка «Сбросить фильтры», при нажатии на которую все выбранные фильтры очищаются.
